Core Platform API

\n \n
\n

AlgorithmsLearningPlatform

\n

\n Main application class that orchestrates the learning experience.\n Manages modules, user progress, and the interactive CLI interface.\n

\n \n
\n

Constructor

\n 
class AlgorithmsLearningPlatform {\n    constructor(options?: PlatformOptions)\n}
\n
\n\n
\n

Methods

\n
\n
async run(): Promise<void>
\n

Starts the interactive learning platform and displays the main menu.

\n
\n \n
\n
async loadModule(moduleName: string): Promise<void>
\n

Loads and executes a specific learning module.

\n
\n Parameters:\n
    \n
  • moduleName - Name of the module to load (e.g., 'arrays', 'trees')
    • \n
\n
\n
\n \n
\n
displayWelcome(): Promise<void>
\n

Shows the welcome screen with platform introduction.

\n
\n \n
\n
showMainMenu(): Promise<string>
\n

Displays the main navigation menu and returns user selection.

\n
\n
\n\n
\n

Properties

\n
    \n
  • userProgress: UserProgress - Current user progress data
    • \n
  • currentLevel: string - Current difficulty level
    • \n
\n
\n\n
\n

Usage Example

\n 
// Create and start the platform\nconst platform = new AlgorithmsLearningPlatform({\n    theme: 'dark',\n    startLevel: 'foundation'\n});\n\n// Launch the interactive experience\nawait platform.run();

Learning Modules API

\n \n
\n

Arrays Module

\n

\n Interactive learning module for array operations and algorithms.\n Uses book organization analogies to teach array concepts.\n

\n \n
\n

Module Structure

\n 
// src/modules/arrays/index.js\nexport default {\n    name: 'Arrays: Organizing Books',\n    difficulty: 1,\n    async run(platform) {\n        // Module implementation\n    },\n    lessons: [\n        'Array Basics',\n        'Indexing and Access',\n        'Iteration Patterns',\n        'Common Operations'\n    ]\n}
\n
